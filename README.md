🚀 Web-Based Code Editor
A sleek, lightweight, and modern browser-based code editor. This project aims to provide a fast environment to write and test code with persistent storage.

✨ Current Features
🎨 Modern UI: A revamped, distraction-free interface for a better coding experience.

💾 Local Persistence: Automatically saves your code to Local Storage, so you never lose your work on refresh.

⚙️ Optimized Logic: Refined core script for faster performance and better file handling.

🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript (ES6+)

Storage: LocalStorage API

🗺️ Roadmap (Coming Soon)
☁️ Cloud Sync: Server-side integration to access your code from any device.

💡 Smart Suggestions: AI-powered syntax hints and auto-complete features.

📂 Multi-file Support: Manage multiple files within the same session.
